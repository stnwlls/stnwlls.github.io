OrderApp

A UIKit restaurant ordering app that fetches menu data, loads item images asynchronously, manages an order cart, submits orders, and confirms estimated prep time.

View Code
Swift UIKit URLSession async/await JSON

What It Does

Users browse restaurant categories, review menu item details, build an order, remove items, submit the order, and view a confirmation screen.

Technical Focus

The app uses a shared menu controller, async network requests, Codable response models, table views, custom cells, and NotificationCenter updates.

Why It Matters

It demonstrates a practical mobile workflow with client-server patterns, structured models, and multi-screen navigation.
