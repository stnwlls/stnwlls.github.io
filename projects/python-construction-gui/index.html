Python Construction GUI

A Python desktop business application for construction service browsing, client quote requests, manager-side service management, and estimate workflows.

View Code
Python Tkinter OOP File I/O Threading

What It Does

Clients can search services and request quotes. Managers can add, edit, delete, search, sort, and review service and quote-request records.

Technical Focus

The project combines GUI screens, custom data models, text-file persistence, sequential search, bubble sort, tax calculations, and threaded logging.

Why It Matters

It demonstrates GUI development, modular Python structure, persistent data, and multi-role business workflows.
