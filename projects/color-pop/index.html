Color Pop

A SpriteKit color-matching game where players tap the balloon matching the written color name while racing against a timer.

View Code
Swift SpriteKit Game State Touch Input Persistence

What It Does

The game randomizes color labels and balloon positions, tracks score, applies timer pressure, saves high scores, and rewards streaks with bonus time.

Technical Focus

Color Pop uses SpriteKit scenes, SKSpriteNode and SKLabelNode objects, SKAction timers, touch handling, particle effects, and saved-game persistence.

Why It Matters

It demonstrates interactive iOS development with responsive game loops, touch input, timers, persistence, and visual feedback.
